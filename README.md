# sora2-browser-tool
A split screen UI for browsing and using free Sora2 AI prompts

Features
- Browse 100+ free Sora 2 AI sites on left pane (configurable in json)
- Disposable mail site on right pane (configurable in json)
- Customizable user files created for custom sites, characters, and prompts
- Customizable list of characters, prompts, and sites saved in json format
- Customizable characters, prompts, and sites with import/export
- Restore user files from core data
- Can change user agent if needed (most times breaks captchas)
- Added preview window and autofill for characters or manual entry in prompts
- Edit prompts using right-click menu
- 4 customizable characters
- Custom and saved window pane values on exit
